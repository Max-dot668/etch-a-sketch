# Etch-a-Sketch Project

A browser-based Etch-a-Sketch/Sketchpad project built for The Odin Project Foundations Course. This project demonstrates fundamental DOM manipulation, Flexbox layout skills, and interactive JavaScript behavior.

## Features

* Dynamically generated grid (default 16x16)
* Hover-based drawing interaction
* Button to resize grid up to 100x100
* Optional extra credit features:

  * Random RGB colors
  * Progressive darkening (10% each pass)

## Technologies Used

* HTML
* CSS (Flexbox for grid layout)
* JavaScript (DOM manipulation & events)

## How to Use

1. Hover your mouse over the grid cells to draw.
2. Click the **Resize Grid** button to enter a new grid size.
3. Experiment with additional color/darkening modes (if implemented).

## Project Setup
